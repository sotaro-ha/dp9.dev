# タイトル

このチュートリアルでは、Next.jsを使用してチュートリアルサイトを作成する方法を学びます。

## セクション

1. Next.jsプロジェクトのセットアップ
2. 必要なパッケージのインストール
3. Markdownファイルの作成
4. ルーティングの設定
5. `getStaticPaths`と`getStaticProps`の設定
6. ナビゲーションの自動生成
