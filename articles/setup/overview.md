## セクション

1. Next.js プロジェクトのセットアップ
2. 必要なパッケージのインストール
3. Markdown ファイルの作成
4. ルーティングの設定
5. `getStaticPaths`と`getStaticProps`の設定
6. ナビゲーションの自動生成
